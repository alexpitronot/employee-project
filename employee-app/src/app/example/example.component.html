Pipes Examples

Today is:

{{today | date}}

Full date is:

{{today | date:'fullDate'}}

The time is:

{{today | date:'shortTime'}}

System currency is:

{{curr | currency}}

Local currency is:

{{curr | currency: 'ILS'}}

Local currency is:

{{curr | currency: 'Shekel ' | uppercase}}

Pi number with no specified formatting:

{{pi | number}}

Pi number with digitsInfo parameter specified:

{{pi | number:'2.1-10'}}

Pi as percent:

{{pi | percent}}

Employee in json format:

{{employee | json}}
